# TP Estadística métodos computacionales

Este repositorio contiene scripts en Python para la descarga, procesamiento y análisis de datos financieros históricos. El proyecto abarca desde el cálculo de estadísticas descriptivas (rendimiento y volatilidad) hasta la optimización de portafolios y la construcción de la Frontera Eficiente.

## Pasos para ejecutar el código (pre-requisito, tener instalado Python 3.x)
### 1. Clonar el repositorio

```bash
git clone [https://github.com/mirkobasar/estadistica_metodos_computacionales.git](https://github.com/mirkobasar/estadistica_metodos_computacionales.git)
cd estadistica_metodos_computacionales
```

### 2. Crear entorno virtual
```bash
python -m venv venv
```

### 3. Activar entorno virtual
```bash
source venv/bin/activate
```

### 4. Instalar dependencias
```bash
pip install -r requirements.txt
```

### 5. Ejecutar script
```bash
python get_ticker.py
```
